task for JDP at VITech

# Run Node.Js server
**1 step - you need to clone this repo**
```
$ git clone [url]

```
**2 step - Build Spring Boot project**
```
mvn clean install

```
**3 step - Run Spring Boot project**
```
mvn spring-boot:run

```
**4 step - install Node**
[Node.js](https://nodejs.org/en/).

```
**5 step - install node modules**
```
npm install

```
**6 step - run Angular**
```
ng serve

```
